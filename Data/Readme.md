# 📊 Data

This project utilizes the **DDXPlus Dataset (English)**. 

> **Note:** Due to its large size (~800 MB), the dataset is not included in this repository.

## 📥 Download Instructions

Please download the dataset from Figshare:

🔗 **[DDXPlus Dataset (English)](https://figshare.com/articles/dataset/DDXPlus_Dataset_English_/22687585)**

## 📂 Setup

Follow these steps to prepare the data for the project:

1. **Download** the dataset from the link provided above.
2. **Extract** the contents of the downloaded archive.
3. **Move** the extracted files into the `data/` directory located in the project root.

### Directory Structure
After extraction, your project directory should look like this:

```text
data/
├── ddxplus/
│   ├── release_conditions.json
│   ├── release_evidences.json
│   └── ... (other dataset files)
└── README.md
